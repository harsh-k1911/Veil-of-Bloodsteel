🩸 Veil of Bloodsteel

A dark, cinematic ninja arsenal marketplace built with pure HTML, CSS, and JavaScript.

“Power is not bought. It is acquired.”

⚔️ Overview

Veil of Bloodsteel is a premium frontend project that transforms a simple e-commerce concept into an immersive underground weapons market.

No generic products.
No basic UI.
Only shadow-forged steel and cinematic design.

🔥 Features
🥷 Curated ninja arsenal (katanas, masks, daggers, cloaks, shurikens)
🛒 Inventory system with localStorage
💰 Indian pricing (₹ format)
🎨 Dark UI with glow + glass effects
🖱️ Interactive cards with hover + 3D tilt
🌌 Cinematic hero with fire background
📱 Fully responsive
🛠 Tech

HTML • CSS • JavaScript (Vanilla)
